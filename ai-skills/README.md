## 📚 已收录的 Skills

### 🔧 核心机制

- [全局自动经验沉淀触发器](.lingma/skills/全局自动经验沉淀触发器.md)
  - **功能**：全程静默监控，自动提炼最优解法
  - **触发条件**：时长 >40s / 试错 ≥3轮 / 同源场景
  - **适用**：所有类型的问题（通用元规则）

### 🌐 中间层通信

- [中间层消息流问题排查](.lingma/skills/中间层消息流问题排查.md)
  - **功能**：两端消息排查法 + 立即重启策略
  - **适用场景**：Cursor ↔ LamaPuppeteer ↔ DeepSeek 通信异常
  - **效率提升**：从 30 分钟 → 30 秒

## 🎯 如何使用

### 在 LamaPuppeteer 中使用

这些 Skills 已经集成到 LamaPuppeteer 项目中，会自动触发：

```bash
cd lama-puppeteer
.\start_dispatcher.bat
```

### 在其他项目中复用

你可以将这些 Skills 复制到其他项目的 `.lingma/skills/` 目录中，享受同样的自动经验沉淀能力。
