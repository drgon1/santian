如何使用

方式 1：直接在 GitHub 查看

点击上方的 Skill 链接，直接在 GitHub 上阅读完整文档。

方式 2：克隆到本地使用

git clone https://github.com/drgon1/santian.git
cd santian/ai-skills/.lingma/skills

方式 3：导入到 Lingma IDE

将整个 Skill 文件夹复制到你的 .lingma/skills/ 目录即可使用。
